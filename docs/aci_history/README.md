# ACI History

This directory records the execution history of Architecture Change Instructions (ACIs) for the TAIG NOPE Portal.

## Purpose

Each ACI represents a discrete, governed change to the repository. Completion reports are stored in `docs/reports/`. This directory provides a chronological index of ACI executions.

## Entries

| ACI | Title | Status | Report |
|-----|-------|--------|--------|
| ACI-001 | Repository Foundation | Complete | [ACI-001-completion-report.md](../reports/ACI-001-completion-report.md) |
| ACI-002 | Data Model Foundation | Complete | [ACI-002-completion-report.md](../reports/ACI-002-completion-report.md) |

## Conventions

- One completion report per ACI in `docs/reports/`
- Reports use the naming pattern `ACI-NNN-completion-report.md`
- Local operator artifacts remain in `nebula_local/` and are never committed
